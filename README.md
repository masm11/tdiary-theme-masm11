# Masm11's plugin and stylesheet

## How to install

```sh
cd misc/plugin/
git clone https://github.com/masm11/tdiary-theme-masm11 masm11
ln -s masm11/plugins/masm11*.rb .
cd ../../theme
ln -s ../misc/plugin/masm11/css masm11
cd masm11
sass masm11.scss masm11.css
```

## How to setup

1. activate plugins:
   - masm11-dow
   - masm11-hljs
   - masm11-theme
2. select theme:
   - masm11
